java验证码思路，简单demo

思路：
生成一个 UUID，将 4 位数验证码与 id 绑定，存入 redis,并设置有效时间。登录时前端传入 UUID，然后你去 redis 查询匹配的验证码，是否存在，是否正确。

说明：
UUID 确保是同一用户操作，避免多人请求，无法确定当前验证码是那个用户的请求
redis 有效时间，删除掉无用的验证码数据
